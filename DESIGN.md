# Design System: ConnectingDocs Front
**Project ID:** 5884243527570635203

## 1. Visual Theme & Atmosphere
A sophisticated, high-authority "Clinical OS" aesthetic. It prioritizes precision and trust, drawing from high-end financial dashboards and modern medical software. The environment is immersive dark mode with sharp, high-contrast elements.

## 2. Color Palette & Roles
* **Deep Space Black (#050505):** Primary page background, absolute dark.
* **Surface Gray (#111111):** Card and section backgrounds for subtle separation.
* **Medical Cyan (#00F0FF):** Primary accent for data highlights, active icons, and high-importance CTAs.
* **Pure White (#FFFFFF):** High-contrast primary text and prominent heading levels.
* **Muted Slate (#888888):** Secondary/Tertiary text for labels and supporting descriptions.
* **Border Slate (#1F1F1F):** Subtle borders for bento grid elements and inputs.

## 3. Typography Rules
* **Family:** Inter or SF Pro Display (System High-Resolution).
* **Headings:** Bold weights, high-contrast, slightly tighter tracking for an authoritative "Tech" feel.
* **Body:** Medium weights for readability, generous line height (1.6) for professional clarity.
* **Data Numbers:** Monospace or clean Sans-Serif for a "Precision Instrument" look.

## 4. Component Stylings
* **Buttons:** Pill-shaped (rounded-full) for action-oriented CTAs. Glossy or high-contrast outline finishes.
* **Bento Cards:** Subtly rounded corners (12px), whisper-soft borders (#1F1F1F), no heavy shadows—depth is created through color layering.
* **Visuals:** Use high-resolution medical/tech textures, glassmorphism for overlays, and clean data visualizations (radar charts).

## 5. Layout Principles
* **Structure:** Clean Bento Grid architecture for information organization.
* **Margins:** Precise, consistent padding (Gaps: 24px, 32px) for an airy but dense professional feel.
* **Hierarchy:** Large HERO typography prioritized, followed by data-driven sub-blocks.
